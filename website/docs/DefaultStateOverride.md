# DefaultStateOverride

`DefaultStateOverride`は、マスコットの待機状態（アイドルモーション）を複数のバリエーションから選択できるようにするコンポーネントです。

## 概要

このコンポーネントを使用することで、マスコットがアクションを実行していない待機状態のときに再生されるアニメーションを、複数のオプションから選択できます。例えば、「立ちポーズ」「座りポーズ」「寝そべりポーズ」などの異なるアイドルモーションを設定し、プレイヤーがメニューから選択できるようになります。選択されたアイドルモーションは、インスタンス内の全プレイヤーで同期されます。

また、各アクション（`ContactAction`、`PhysBoneTriggerAction`、`RandomTriggerAction`など）には、特定のアイドルモーション時にのみ実行可能に設定するオプションがあります。これにより、「座っているときだけ反応する」「立っているときだけ動く」といった細かい制御が可能になります。

## パラメータ

インスペクターでは、以下のパラメータを持つアイドルモーションをリストで管理します。

| パラメータ | 型 | 説明 |
| :--- | :--- | :--- |
| **Override State Name** | `string` | アイドルモーションの名前。メニューUIで表示されます。 |
| **Override Animation Clip** | `AnimationClip` | このアイドルモーションで使用するアニメーションクリップ。 |

## 使用方法

1. `DefaultStateOverride`コンポーネントをマスコットのオブジェクトに追加します。
2. インスペクターで「Default State Overrides」リストに、必要な数のアイドルモーションを追加します。
3. 各モーションに名前を設定し、対応するアニメーションクリップを割り当てます。
4. ワールド内で、メニューUI経由でアイドルモーションを切り替えることができます。

## 注意事項

- アニメーションクリップは、Animatorコントローラーのレイヤー0（メインレイヤー）で再生されます。
- ステート名は、アニメーションコントローラー内のステート名と対応している必要があります。
- リストの最初のモーション（インデックス0）がデフォルトのアイドルモーションとして使用されます。

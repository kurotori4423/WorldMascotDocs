# WorldMascotDressUp

`WorldMascotDressUp`は、マスコットの着せ替え（衣装変更）機能を提供するコンポーネントです。

## 概要

このコンポーネントを使用することで、マスコットの見た目を複数のプリセットから選択して変更できます。各プリセットには専用のアニメーションクリップを設定し、衣装の切り替えをアニメーションで制御します。選択された衣装は、インスタンス内の全プレイヤーで同期されます。

## パラメータ

インスペクターでは、以下のパラメータを持つ着せ替えプリセットをリストで管理します。

| パラメータ | 型 | 説明 |
| :--- | :--- | :--- |
| **Preset Name** | `string` | プリセットの名前。メニューUIなどで表示されます。 |
| **DressUp Animation Clip** | `AnimationClip` | このプリセットで使用するアニメーションクリップ。衣装の表示/非表示などを制御します。 |

## 使用方法

1. `WorldMascotDressUp`コンポーネントをマスコットのオブジェクトに追加します。
2. インスペクターで「DressUp Presets」リストに、必要な数のプリセットを追加します。
3. 各プリセットに名前を設定し、対応するアニメーションクリップを割り当てます。
4. ワールド内で、UI経由でプリセットを切り替えることができます。

## 注意事項

- アニメーションクリップは、Animatorコントローラー内で着せ替えステート（DressUp State）として使用されます。
- プリセット名は、アニメーションコントローラー内のステート名と対応している必要があります。
